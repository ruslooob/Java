# Access Modifiers

|  Modifier   | Class  | Package | Subclass | World |
|:-----------:|:------:|:-------:|:--------:|:-----:|
| public      |    +   |    +    |     +    |   +   |
| protected   |    +   |    +    |     +    |   -   |
| no modifier |    +   |    +    |     -    |   -   |
| private     |    +   |    -    |     -    |   -   |
